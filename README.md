# 📊 Advanced Excel: Data Lookup & Analysis Dashboard

## 🚀 Project Overview
This project showcases a professional **Dynamic Dashboard** built in Microsoft Excel. The goal of this project was to move beyond static data and create a system that allows for instant data retrieval and automated summarization. 

As an aspiring **Data Analyst**, I focused on accuracy, speed, and a clean user interface.

---

## 🛠️ Key Features & Skills Applied

### 1. Dynamic Search System (Lookups)
I implemented advanced lookup functions to search through the database instantly:
* **VLOOKUP & XLOOKUP:** To fetch specific employee details vertically.
* **HLOOKUP:** To retrieve data from horizontal tables.
* **Formula Documentation:** Displayed formulas using text formatting to show the logic behind the results.

### 2. Data Insights (Pivot Tables)
* Created **Pivot Tables** to summarize large datasets.
* Analyzed **Department-wise Salary Distribution** to provide quick executive insights.

### 3. Professional UI/UX in Excel
* **Header Design:** Used Merge & Center with custom color themes for a professional look.
* **Dashboard Feel:** Removed gridlines and organized tables into logical sections (Master Data vs. Dynamic Search).

## 📈 Tools Used
* **Microsoft Excel** (Advanced Formulas, Pivot Tables, Formatting)
* **Data Analysis Concepts** (Summarization, Retrieval)
